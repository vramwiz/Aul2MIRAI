# フィルター効果学習

この文書は、フィルター名とパラメーターが実際の映像へ与える変化を、基準素材、設定値、キャプチャ比較とともに記録する。生成用エイリアスと値形式は[`ALIAS_CATALOG.md`](ALIAS_CATALOG.md)、操作方法は[`AI_USAGE.md`](AI_USAGE.md)、時系列は[`HISTORY.md`](HISTORY.md)を正本とする。

## 共通実験方法

1. 色帯、グラデーション、市松模様、細線、輪郭、文字を含む決定論的な基準画像を一時PNGとして作る。
2. 基準画像を参照する画像オブジェクトと対象フィルターを、1個のエイリアスとして生成する。
3. フィルター以外の条件とカーソルフレームを固定する。
4. 対象パラメーターだけを段階的に変更し、各状態を`get_current_frame_image`で取得する。
5. 目視差と画素差を記録する。時間依存効果では開始・中間・終了など複数フレームを使う。

## 適用方式1: オブジェクト自身のフィルター

### ぼかし

- 確認環境: AviUtl2 v2.1.0、1920×1080、30 fps
- 基準画像: 640×360の色帯、白黒グラデーション、市松模様、1px細線、赤い楕円、白文字
- 生成構成: `画像ファイル`、`標準描画`、`ぼかし`
- 配置: レイヤー0、1300～2100フレーム
- 比較フレーム: 1500
- 固定値: `縦横比=0.00`、`光の強さ=0.0`、`サイズ固定=0`
- 比較値: `範囲=5`、`範囲=50`

確認結果:

- 生成直後の詳細取得で、画像素材パス、3効果の順序、`ぼかし.範囲=5`を確認した。
- `範囲=5`では、色境界、細線、楕円輪郭、文字が軽く軟化した。
- `範囲=50`では、細線、市松模様、文字の判別性が大きく低下し、色境界が広範囲に混ざった。
- 同じフレームの2画像を比較すると、2,073,600画素中338,500画素が変化した。変化率は16.3243%、RGB各成分の平均絶対差は8.6565、最大成分差は255だった。
- 変化領域の外接矩形は`(590,310)～(1329,769)`で、中央に配置した画像とぼかしによる外側への広がりに対応した。背景の離れた領域には変化がなかった。
- 以上から、画像がオブジェクト内部の`ぼかし`を通り、`範囲`の変更がレンダリング結果へ反映されることを確認した。

未確認:

- `範囲=0`との完全な無効果比較
- `縦横比`、`光の強さ`、`サイズ固定`
- オブジェクト区間外
- グループ制御方式と独立フィルター方式

### 色調補正

- 確認環境: AviUtl2 v2.1.0、1920×1080、30 fps
- 基準画像: `ぼかし`と同じ640×360の基準PNG
- 生成構成: `画像ファイル`、`標準描画`、`色調補正`
- 配置: レイヤー0、2200～3000フレーム
- 比較フレーム: 2400
- 中立値: `明るさ=100.00`、`コントラスト=100.00`、`色相=0.00`、`輝度=100.00`、`彩度=100.00`、`飽和する=1`

確認結果:

- 生成直後の詳細取得で、画像素材パス、3効果の順序、中立値6項目を確認した。
- `明るさ=50`では黒を保ったまま全体のRGB値がほぼ比例して暗くなった。白は中間灰色になり、100を中立とする光量倍率として作用した。
- `コントラスト=50`では黒が明るく、白が暗くなり、色帯とグラデーションが中間灰色へ圧縮された。
- `色相=120`では色帯と赤い楕円の色が回転し、黒、白、グレースケールはほぼ維持された。アンチエイリアス境界の色成分にも回転が現れた。
- `輝度=50`では白、灰色、市松模様などの無彩色が暗くなった一方、純色の色帯と赤い楕円は`明るさ=50`ほど単純には暗くならなかった。RGB一括倍率ではなく、輝度成分へ作用すると判断できる。
- `彩度=0`では色帯と赤い楕円がグレースケール化され、無彩色領域は維持された。
- `明るさ=120`と`200`で`飽和する=1/0`を比較したが、この不透明RGB基準画像ではどちらも画素差0だった。チェック値の書込みと再読取は成功したが、作用条件は特定できなかった。
- 各比較後に変更項目を中立値へ戻し、最終状態が中立値6項目であることを確認した。

中立画像との画素比較:

| 変更 | 変化画素率 | RGB成分の平均絶対差 | 最大成分差 |
| --- | ---: | ---: | ---: |
| `明るさ=50` | 9.5769% | 5.3438 | 128 |
| `コントラスト=50` | 11.1063% | 6.3730 | 64 |
| `色相=120` | 6.9380% | 3.1968 | 255 |
| `輝度=50` | 9.5769% | 4.0249 | 127 |
| `彩度=0` | 6.9380% | 2.2838 | 226 |
| `明るさ=120` | 9.2365% | 4.1773 | 51 |
| `明るさ=200` | 9.2365% | 19.1006 | 255 |

- すべての変化領域は基準画像の表示範囲`(640,360)～(1279,719)`内に収まり、背景には変化がなかった。
- `飽和する=1/0`の相互比較は、`明るさ=120`と`200`の両条件で2,073,600画素すべて一致した。

未確認:

- `明るさ`、`コントラスト`、`色相`、`輝度`、`彩度`の上限・下限付近と複合設定
- 透過・半透明素材や複数調整を使った`飽和する`の作用条件
- グループ制御方式と独立フィルター方式
